## Как сохранить файлы

### Вариант 1: Если вы экспортировали из Figma

1. **Экспортированные файлы из Figma:**
   - Откройте папку с экспортированными файлами
   - Переименуйте их в:
     - `person-1.png`
     - `person-2.png`
     - `person-3.png`
     - `person-4.png`

2. **Скопируйте файлы в проект:**
   
   **Для WordPress темы:**
   ```
   wordpress-theme/assets/images/person-1.png
   wordpress-theme/assets/images/person-2.png
   wordpress-theme/assets/images/person-3.png
   wordpress-theme/assets/images/person-4.png
   ```
   
   **Для статической версии:**
   ```
   images/person-1.png
   images/person-2.png
   images/person-3.png
   images/person-4.png
   ```

### Вариант 2: Если изображения уже в проекте

Если вы уже скопировали изображения в папки проекта, просто проверьте, что они имеют правильные имена.

### Вариант 3: Перетаскивание в VS Code

1. Откройте папку `wordpress-theme/assets/images/` в VS Code
2. Перетащите файлы из папки загрузок
3. Переименуйте их, если нужно

## Проверка после размещения

После того, как файлы размещены, проверьте их:

```powershell
# Проверка хешей файлов
Get-FileHash images\person-*.png | Format-Table

# Проверка размеров файлов
Get-ChildItem images\person-*.png | Select-Object Name, Length | Format-Table
```

Хеши должны быть **разными** для каждого файла!
