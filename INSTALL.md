# Инструкция по установке WordPress локально

## Быстрая установка с Docker

### Шаг 1: Установите Docker Desktop
Скачайте и установите Docker Desktop с официального сайта: https://www.docker.com/products/docker-desktop

### Шаг 2: Запустите WordPress
В корневой папке проекта выполните команду:
```bash
docker-compose up -d
```

### Шаг 3: Откройте WordPress
Откройте браузер и перейдите по адресу: http://localhost:8080

### Шаг 4: Завершите установку WordPress
1. Выберите язык (русский)
2. Заполните информацию о сайте:
   - Название сайта: Simple UP
   - Имя пользователя: admin
   - Пароль: (придумайте надежный пароль)
   - Email: ваш email

### Шаг 5: Активируйте тему
1. Перейдите в `Внешний вид → Темы`
2. Найдите тему "Simple UP Theme"
3. Нажмите "Активировать"

### Шаг 6: Используйте шорткоды
Создайте новую страницу и добавьте шорткоды:
- `[simple_up_hero]`
- `[simple_up_features]`
- `[simple_up_process]`

## Установка без Docker

### Вариант 1: Local by Flywheel (Рекомендуется)

1. **Скачайте Local by Flywheel**
   - Перейдите на https://localwp.com/
   - Скачайте и установите приложение

2. **Создайте новый сайт**
   - Запустите Local
   - Нажмите "Create a new site"
   - Выберите имя сайта: "Simple UP"
   - Выберите среду: "Preferred" (последняя версия WordPress)
   - Создайте учетную запись администратора

3. **Установите тему**
   - После создания сайта нажмите "Open Site Shell"
   - Скопируйте папку `wordpress-theme` в `wp-content/themes/`
   - Или используйте FTP/SFTP для загрузки темы

4. **Активируйте тему**
   - Перейдите в админ-панель WordPress
   - `Внешний вид → Темы`
   - Активируйте "Simple UP Theme"

### Вариант 2: XAMPP

1. **Установите XAMPP**
   - Скачайте с https://www.apachefriends.org/
   - Установите XAMPP

2. **Скачайте WordPress**
   - Скачайте WordPress с https://ru.wordpress.org/
   - Распакуйте в папку `C:\xampp\htdocs\simple-up`

3. **Создайте базу данных**
   - Откройте http://localhost/phpmyadmin
   - Создайте новую базу данных: `simple_up`

4. **Установите WordPress**
   - Откройте http://localhost/simple-up
   - Завершите установку WordPress
   - Используйте базу данных: `simple_up`
   - Имя пользователя БД: `root`
   - Пароль БД: (оставьте пустым)

5. **Установите тему**
   - Скопируйте папку `wordpress-theme` в `C:\xampp\htdocs\simple-up\wp-content\themes\`
   - Переименуйте в `simple-up-theme`

6. **Активируйте тему**
   - Перейдите в админ-панель WordPress
   - `Внешний вид → Темы`
   - Активируйте "Simple UP Theme"

### Вариант 3: WAMP

1. **Установите WAMP**
   - Скачайте с https://www.wampserver.com/
   - Установите WAMP

2. **Следуйте инструкциям для XAMPP**
   - Замените `C:\xampp\htdocs` на `C:\wamp\www`

## Настройка меню

1. Перейдите в админ-панель WordPress
2. `Внешний вид → Меню`
3. Создайте новое меню
4. Добавьте пункты меню (Главная, О нас, Услуги, Контакты)
5. Назначьте меню в позицию "Основное меню"
6. Сохраните изменения

## Использование шорткодов

### В редакторе блоков (Gutenberg):
1. Создайте новую страницу
2. Добавьте блок "Шорткод"
3. Вставьте один из шорткодов:
   ```
   [simple_up_hero title="Добро пожаловать" description="Описание" button_text="Начать" button_link="#"]
   [simple_up_features title="Наши возможности" columns="4"]
   [simple_up_process title="Как мы работаем"]
   ```

### В классическом редакторе:
1. Создайте новую страницу
2. Переключитесь в текстовый режим
3. Вставьте шорткоды напрямую в содержимое

### В шаблоне темы:
Отредактируйте `wordpress-theme/index.php` и используйте:
```php
<?php echo do_shortcode('[simple_up_hero]'); ?>
```

## Проверка работы

1. Откройте главную страницу сайта
2. Убедитесь, что все три блока отображаются корректно
3. Проверьте адаптивность на мобильных устройствах
4. Проверьте работу меню

## Устранение неполадок

### Стили не загружаются:
- Проверьте, что файл `wordpress-theme/assets/css/style.css` существует
- Очистите кеш браузера (Ctrl+F5)
- Проверьте права доступа к файлам

### Шорткоды не работают:
- Убедитесь, что тема активирована
- Проверьте, что файл `wordpress-theme/inc/shortcodes.php` подключен в `functions.php`
- Проверьте ошибки PHP в логах

### Меню не отображается:
- Создайте меню в `Внешний вид → Меню`
- Назначьте меню в позицию "Основное меню"
- Или используйте fallback меню из темы
