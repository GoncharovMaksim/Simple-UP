# Быстрый старт

## Шаг 1: Предпросмотр верстки

Откройте файл `index.html` в браузере, чтобы увидеть верстку без WordPress.

## Шаг 2: Установка WordPress

### Вариант A: Docker (самый простой)

1. Установите Docker Desktop
2. Выполните команду:
   ```bash
   docker-compose up -d
   ```
3. Откройте http://localhost:8080
4. Завершите установку WordPress

### Вариант B: Local by Flywheel

1. Установите Local by Flywheel
2. Создайте новый сайт
3. Скопируйте папку `wordpress-theme` в `wp-content/themes/`

## Шаг 3: Активация темы

1. Войдите в админ-панель WordPress
2. Перейдите в `Внешний вид → Темы`
3. Активируйте "Simple UP Theme"

## Шаг 4: Использование шорткодов

### Создайте новую страницу:

1. Перейдите в `Страницы → Добавить новую`
2. Добавьте блок "Шорткод"
3. Вставьте шорткоды:

```
[simple_up_hero title="Добро пожаловать в Simple UP" description="Мы создаем современные решения для вашего бизнеса." button_text="Начать работу" button_link="#"]

[simple_up_features title="Наши возможности" columns="4"]

[simple_up_process title="Как мы работаем"]
```

4. Опубликуйте страницу

## Готово!

Ваша страница с версткой по макету Figma готова!

## Дополнительная настройка

### Настройка меню:

1. `Внешний вид → Меню`
2. Создайте меню
3. Добавьте пункты
4. Назначьте в позицию "Основное меню"

### Изменение контента:

Отредактируйте файл `wordpress-theme/inc/shortcodes.php`, чтобы изменить содержимое по умолчанию для шорткодов.
