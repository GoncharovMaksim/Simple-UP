# Как вставить изображения из Figma

## Шаг 1: Сохраните изображения на компьютер

Если вы скопировали изображения из Figma:
1. Откройте любой графический редактор (Paint, Photoshop, GIMP) или просто сохраните через браузер
2. Сохраните каждое изображение с правильным именем:
   - `person-1.png` - человек с вопросительными знаками (карточка 1)
   - `person-2.png` - спящий человек с "Zz" (карточка 3)
   - `person-3.png` - обеспокоенный человек с крестиками (карточка 5)
   - `person-4.png` - человек в блоке решения

## Шаг 2: Скопируйте файлы в проект

### Вариант A: Перетаскивание в VS Code

1. Откройте папку `images` в VS Code (в левой панели)
2. Перетащите файлы из папки загрузок в папку `images`
3. Убедитесь, что файлы называются правильно:
   - `person-1.png`
   - `person-2.png`
   - `person-3.png`
   - `person-4.png`

4. Затем скопируйте те же файлы в папку `wordpress-theme/assets/images`

### Вариант B: Через проводник Windows

1. Откройте папку проекта: `d:\vscode\tests-HHru\Simple-UP\images`
2. Скопируйте файлы `person-1.png`, `person-2.png`, `person-3.png`, `person-4.png` в эту папку
3. Откройте папку: `d:\vscode\tests-HHru\Simple-UP\wordpress-theme\assets\images`
4. Скопируйте те же файлы в эту папку

### Вариант C: Через командную строку

Если файлы находятся, например, в папке `C:\Users\ВашеИмя\Downloads`:

```powershell
# Копирование в images
Copy-Item "C:\Users\ВашеИмя\Downloads\person-1.png" -Destination "d:\vscode\tests-HHru\Simple-UP\images\person-1.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-2.png" -Destination "d:\vscode\tests-HHru\Simple-UP\images\person-2.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-3.png" -Destination "d:\vscode\tests-HHru\Simple-UP\images\person-3.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-4.png" -Destination "d:\vscode\tests-HHru\Simple-UP\images\person-4.png" -Force

# Копирование в wordpress-theme/assets/images
Copy-Item "C:\Users\ВашеИмя\Downloads\person-1.png" -Destination "d:\vscode\tests-HHru\Simple-UP\wordpress-theme\assets\images\person-1.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-2.png" -Destination "d:\vscode\tests-HHru\Simple-UP\wordpress-theme\assets\images\person-2.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-3.png" -Destination "d:\vscode\tests-HHru\Simple-UP\wordpress-theme\assets\images\person-3.png" -Force
Copy-Item "C:\Users\ВашеИмя\Downloads\person-4.png" -Destination "d:\vscode\tests-HHru\Simple-UP\wordpress-theme\assets\images\person-4.png" -Force
```

## Шаг 3: Проверка

После копирования проверьте, что файлы разные:

```powershell
Get-FileHash images\person-*.png | Format-Table
```

Хеши должны быть **разными** для каждого файла!

## Важно!

- Файлы должны называться точно: `person-1.png`, `person-2.png`, `person-3.png`, `person-4.png`
- Файлы должны быть в формате PNG
- Файлы должны быть размещены в двух местах:
  - `images/` (для статической версии)
  - `wordpress-theme/assets/images/` (для WordPress темы)
