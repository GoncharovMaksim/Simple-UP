# Simple UP - Верстка и WordPress интеграция

## Описание проекта

Проект содержит верстку 3 экранов по макету Figma и интеграцию с WordPress через шорткоды.

## Структура проекта

```
.
├── index.html              # HTML верстка для предпросмотра
├── css/
│   └── style.css          # Стили для верстки
├── wordpress-theme/       # WordPress тема
│   ├── style.css          # Заголовок темы
│   ├── functions.php      # Функции темы
│   ├── index.php          # Основной шаблон
│   ├── header.php         # Шапка сайта
│   ├── footer.php         # Подвал сайта
│   ├── inc/
│   │   └── shortcodes.php # Шорткоды WordPress
│   └── assets/
│       └── css/
│           └── style.css  # Стили темы
└── README.md
```

## Верстка экранов

### БЛОК 1: Header + Hero Section
- Навигационное меню
- Заголовок и описание
- Кнопка призыва к действию

### БЛОК 6-7: Features Section
- Секция с возможностями/особенностями
- 4 карточки с иконками

### БЛОК 8: Process Section
- Секция с процессом работы
- Два столбца: элементы процесса и шаги

## Установка WordPress локально

### Вариант 1: Использование Local by Flywheel

1. Скачайте и установите [Local by Flywheel](https://localwp.com/)
2. Создайте новый сайт
3. Скопируйте папку `wordpress-theme` в `wp-content/themes/`
4. Активируйте тему в админ-панели WordPress

### Вариант 2: Использование XAMPP/WAMP

1. Установите XAMPP или WAMP
2. Скачайте WordPress с официального сайта
3. Разместите WordPress в папке `htdocs` (XAMPP) или `www` (WAMP)
4. Создайте базу данных через phpMyAdmin
5. Запустите установку WordPress
6. Скопируйте папку `wordpress-theme` в `wp-content/themes/`
7. Активируйте тему в админ-панели WordPress

### Вариант 3: Использование Docker

1. Установите Docker Desktop
2. Используйте готовый Docker Compose файл (см. ниже)

## Использование шорткодов

### Шорткод Hero Section
```
[simple_up_hero title="Заголовок" description="Описание" button_text="Текст кнопки" button_link="ссылка"]
```

### Шорткод Features Section
```
[simple_up_features title="Заголовок" columns="4"]
```

### Шорткод Process Section
```
[simple_up_process title="Заголовок"]
```

## Примеры использования в WordPress

### В редакторе страниц/постов:
1. Перейдите в редактор страницы или поста
2. Добавьте блок "Шорткод" или используйте редактор блоков
3. Вставьте один из шорткодов:
   - `[simple_up_hero]`
   - `[simple_up_features]`
   - `[simple_up_process]`

### В шаблоне темы:
```php
<?php echo do_shortcode('[simple_up_hero title="Мой заголовок"]'); ?>
```

## Настройка меню

1. Перейдите в `Внешний вид → Меню`
2. Создайте новое меню
3. Добавьте пункты меню
4. Назначьте меню в позицию "Основное меню"

## Стили

Все стили находятся в `wordpress-theme/assets/css/style.css` и автоматически подключаются через `functions.php`.
